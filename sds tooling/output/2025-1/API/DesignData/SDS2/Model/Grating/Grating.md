##### Remarks

To set a grade look for the Grade property on the derived class. You'll need to choose a grade from an appropriate grade list for that material type. Which can be found in a static property, on that class, called AvailableSteelGrades.

##### Exceptions

| Type                                                                           | Condition                                                                                                                                                                                          |
| ------------------------------------------------------------------------------ | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [InvalidValueException](DesignData.SDS2.Exceptions.InvalidValueException.html) | Attempting to set a grade not in this type of material's grade list will throw an InvalidValueException. This is determined by searching that grade list for a grade which is equal (operator ==). |
