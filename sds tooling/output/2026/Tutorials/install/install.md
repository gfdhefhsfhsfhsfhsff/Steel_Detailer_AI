# Install a development environment for .NET and SDS2

You have a choice of environments and languages. This tutorial will show how to use the C# language and .NET Core in Visual Studio Code.

First, [install VS Code](https://code.visualstudio.com/docs/setup/setup-overview) and [configure it for C# development](https://code.visualstudio.com/docs/languages/dotnet) as detailed by Microsoft. These instructions from Microsoft include building a "Hello World" project.

Second, install SDS2\. This will install the .NET assemblies and the native DLL in the installation directory.
